The Concept

  • I recently bought a Gear S3 Classic smartwatch. When I paired it with my iPhone, I discovered that I could receive messages but not send them.




  • I set out to build an application to allow me to send messages from my watch over WiFi.




  • This code is a native messaging application for the Gear S3 Smartwatch and a server for sending messages.
