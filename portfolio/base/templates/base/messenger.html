Messenger App for the Gear S3 Smartwatch

(https://github.com/Topazoo/Gear_S3_Wifi_Messaging)

The Concept

  • I recently bought a Gear S3 Classic smartwatch. When I paired it with my iPhone, I discovered that I could receive messages but not send them. I set out to build an application to allow me to send messages from my watch.


  • The code below is a server for sending messages and a native messaging application for the Gear S3 Smartwatch.

    • Server Demo






    Sending Text Messages from The Server

  • The watch has no built in text messaging server, so I built this server to send messages to phones using the SMTP email protocol. The server sends the message from a Gmail address in config.ini.

  • In the code below, messages from the client are received as POST requests where they are parsed and sent as text messages. See the demo above!

  • The hard part was writing a client that runs natively on the watch. The code is below, written in C for the Tizen Wearable 3.0+ OS.

  • It uses the Tizen HTTP library to get text from the watch and transmit it to the messaging server when a button is pressed.
