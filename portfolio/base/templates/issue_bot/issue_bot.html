The Concept

Every subreddit on Reddit has its own rules, moderators and regular posters. As a result, each represents its own community in which certain political issues are more salient.



Understanding which issues are politically salient to individual communities is a powerful tool. It can be used to target and prime messages.



This code analyzes subreddit comments to determine the issues most salient to that community.
